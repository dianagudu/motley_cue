motley cue¶

This is a service for mapping OIDC identities to local identities.

Mapper Oidc To Local idEntitY with loCal User managEment

For SSH integration, see SSH OIDC.

For client-side software, see mccli.

Compatibility¶

motley_cue works with Python 3, and only on Linux.
